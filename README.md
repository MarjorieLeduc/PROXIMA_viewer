# Content
PROXIMA_viewer.R script allows to custom plots made using report button of PROXIMA.R.

## Requirements
R4.4.0 or higher
RStudio 2024.04.1 Build 748 or higher

## Use
To use PROXIMA_viewer.R, open it in Rstudio, then clic Run app (right up). After this you have to load .rds file generated by [PROXIMA.R script](https://github.com/MarjorieLeduc/PROXIMA) when you clic on report button, then clic apply colors button, after what you will be able to visualyse and customise all plots.
Think to check, only for the first time, if you have all packages.
